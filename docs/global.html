SlashDBProvider(param0)

Custom React component for build in use of Provider from createContext. Use: ....children by defining SlashDBProvider element in project we can pass params to it which will be accessable thoughout project. See SlashDBConsumer for more info on how to use params passed. I.e. Wrap App component so that context data is avaliable throughout application.
Parameters:
Name Type Description
param0 Object holds two parameters: baseUrl (hostname/IP of SlashdDB instance to be accessed, e.g. https://demo.slashdb.com) and setUpOptions. setUpOptions is an object of key value pairs. 'apiKey' is the only key that can be set currently of development.
Properties
Name Type Description
baseUrl string URL of SlashDB instance
setUpOptions Object parameters for connection { apiKey: "somevalue" }
